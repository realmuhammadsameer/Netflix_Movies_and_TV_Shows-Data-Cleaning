# Task 1: Data Cleaning and Preprocessing

## Dataset
Netflix Movies and TV Shows Dataset

## Objective
Clean and preprocess the raw dataset by handling missing values, duplicates, inconsistent formats, and data types.

## Steps Performed

1. Loaded the dataset using Pandas.
2. Checked missing values using isnull().
3. Filled missing values in director, cast, country, rating, and duration columns.
4. Removed rows with invalid or missing date_added values.
5. Removed duplicate records.
6. Standardized column names.
7. Converted date_added to datetime format.
8. Verified data types.
9. Exported the cleaned dataset.

## Files Included

- netflix_titles.csv (Raw Dataset)
- netflix_titles_cleaned.csv (Cleaned Dataset)
- task1_cleaning.ipynb (Cleaning Process)
